# Casual To-Do App

This is a simple casual To-Do application built with Next.js. It allows users to add, toggle, and delete tasks with persistence using localStorage.

## Features

- Add new tasks to your to-do list.
- Mark tasks as completed or incomplete.
- Delete tasks from the list.
- Tasks are saved in localStorage for persistence.
- Clean and responsive UI styled with Tailwind CSS.

## Getting Started

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.

You can start editing the page by modifying `src/app/page.js`. The page auto-updates as you edit the file.
